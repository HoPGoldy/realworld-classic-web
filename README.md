# realworld-classic-web

早期 web 风格的 realworld 实现，基于 react 开发，在线预览地址：[realworld-classic-web](https://hopgoldy.github.io/realworld-classic-web/)。

## 启动

使用: `node 12.16.1+`, `yarn`

```bash
# 安装依赖
yarn

# 启动本地开发服务
yarn start

# 打包
yarn build
```
